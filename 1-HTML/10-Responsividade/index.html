O que é responsividade?

É a tecnica de adaptar uma página web para vários dispositivos diferentes!

Aplicando o media query

Mude a cor pelo tamanho da tela!

Estes breakpoints são utilizaros frequentemente

  • 600px e abaixo → Celular
  • 768px até 600px → Tablets
  • 992px até 768px → Mini laptops
  • 992px e acima → Laptops e desktop

É comum em projetos profissionais utilizamos este valores para desenvolvermos para diferentes dispositivos

landscape:

Para alterar o CSS apenas para usuarios com landscape, precisamos de um atributo chamado orientation com o valor landscape, na media query, o código é ativado quando a orientação é ativada!

Vire o aparelho para mostrar a mensagem em landscape!
